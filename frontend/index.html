XGBoost (eXtreme Gradient Boosting)

Краткая характеристика: XGBoost — ансамблевый метод машинного обучения, реализующий идею градиентного бустинга. В процессе обучения последовательно строятся слабые модели (обычно деревья решений), каждая из которых корректирует ошибки предыдущих.

Используемые параметры:

    n_estimators — количество деревьев в ансамбле. Рекомендуемые значения: 50–300.

    max_depth — максимальная глубина каждого дерева. Рекомендуемые значения: 3–10.

    learning_rate — скорость обучения. Рекомендуемые значения: 0,01–0,3.

KNN (K-Nearest Neighbors)

Краткая характеристика: KNN — непараметрический метод, основанный на принципе ленивого обучения. Классификация объекта осуществляется на основе анализа k ближайших к нему объектов обучающей выборки.

Используемые параметры:

    n_neighbors — количество ближайших соседей. Рекомендуемые значения: 3–15.

    weights — стратегия взвешивания: uniform (равные веса) или distance (вес обратно пропорционален расстоянию).
